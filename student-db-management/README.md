# Student DB Management


Full-stack app: React + TypeScript + Tailwind (frontend) and Flask + MongoDB (backend).


## Setup


### Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev`


### Backend
1. `cd backend`
2. Create and activate virtualenv
3. `pip install -r requirements.txt`
4. Add `.env` with `MONGO_URI` string
5. `python app.py`
